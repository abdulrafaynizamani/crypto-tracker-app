# Crypto Tracker App

A real-time cryptocurrency tracking mobile application built with Flutter, designed using Clean Architecture principles.

## Features
* **Real-time Data:** Fetches live cryptocurrency market prices using REST APIs.
* **Offline Support:** Caches data locally using Hive, allowing the app to work seamlessly without an internet connection.
* **Search Functionality:** Fast and optimized search feature to filter specific coins easily.
* **Price Graphs:** Displays interactive historical price trends using the FL Charts library.
* **Smooth State Management:** Uses Cubit for efficient and predictable state changes.

## Tech Stack
* **Framework:** Flutter / Dart
* **Architecture:** Clean Architecture
* **State Management:** Cubit (flutter_bloc)
* **Local Storage:** Hive
* **Networking:** REST APIs
* **UI & Visuals:** FL Charts
